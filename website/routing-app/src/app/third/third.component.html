Experiments

 

Using this framework and the simulation we built (see Implementation), we performed experiments in which the three types of communication were compared. We first created a 'no communication' baseline that tells us how often a game will be won if there is no communication between players. Then the absolute and relative communications were run with 1 to 3 communications for both types.

With this setup, the simulation was run 1000 times with each setting and we recorded the number of games won and the average number of removed worlds. How many of the games are won gives a good insight into what the best communication strategy is when playing the game. Additionally, the average number of removed worlds says something about how much information is given to the other player with an announcement.

The results of these experiments can be found in the table below.

Type of announcement Number of announcements Ratio won Averge number of worlds reduced per announcement
None 0 0.540 0
Relative 1 0.557 179
2 0.610* 133
3 0.582* 113
Absolute 1 0.552 208
2 0.599* 186
3 0.598* 185
*significant difference with the no communications situation

The table shows that for both absolute and relative communications there is a significant difference with the no communication situation when at least 2 announcements are made. This can be explained by looking at how announcements work in our model. When an absolute or relative announcement is made it reduces the number of possible cards the other player can have. This means it becomes easier to decide which stack to put a card on. There is no significant difference between absolute and relative announcements, which is likely due to using a deck with only 10 cards. It is hard to say which announcement would be better in the real setting with 100 cards, because while the relative announcement divides the set in two, the absolute announcement can potentially remove all cards except for the top three cards. In some cases the relative announcements would be better, while in other cases the absolute announcements are better.

The results also show that for both types of announcements the average number of removed worlds is higher when 1 announcement is made than when 2 or 3 announcements are made. The simple reason for this is that once the first announcement is made, there is only a small set of possible cards left, which means there is not much to remove for the second or third announcement. From this we can make the, somewhat obvious, conclusion that more announcements lead to a more informed and better decision.
