Implementation

 

We have created a program that simulates playing The Game, both for analytical purposes as well as graphical purposes. The following image shows the graphical simulation which aims to help visualize what each player announces, and what the effect of these announcements is on the game. In this simulation, you are free to choose what the announcement type the user would like to be used (absolute, relative or none). Furthermore, the number of announcements that a player can make in a turn can be specified (1, 2 or 3 announcements).

The analytical simulation variant differs from the graphical option in that there is no visualisation. This analytical variant is used to perform our experiments with, as explained in the experiments section.
